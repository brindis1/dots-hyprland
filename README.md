## Dependencies

This configuration setup requires the following applications to be installed for full functionality:

- rofi: for menu and selection UI elements
- rofimoji: rofi emoji menu
- awww: for wallpaper display and transitions
- playerctl: for interacting with media players like Spotify
- pavucontrol: needed for waybar volume icon
- nm-applet: needed for waybar wifi icon
- spotify: for Spotify media controls (optional, used in Waybar script)
